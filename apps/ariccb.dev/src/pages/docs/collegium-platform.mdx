Collegium is a multi-tenant construction management platform that connects owners, consultants, contractors, and vendors. I was the first full-time developer, translating founder vision into a production-ready app with real-time collaboration, BIM ingestion, financial reporting, and granular permissions.

---

## Highlights

- Architected **Next.js (App Router) + Firestore** data models with **TanStack Query** caching and optimistic updates for sub-second perceived latency.
- Integrated **Autodesk APS** to ingest 50k+ BIM elements, enabling interactive model views and automated work-package scopes.
- Built reusable UI components (data tables, filters, bid workflows) with MUI Component Framework, and later transitioned to creating new custom components with **Tailwind + shadcn/ui + Radix**. Utilized Framer Motion for purpose-built animations for clarity and UX polish.
- Implemented role-based access and permission helpers to keep multi-side workflows (owners/consultants/contractors) aligned and secure.
